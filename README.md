# 💰 PromptSack

### Stop losing your genius. Toss it in the sack.

**PromptSack** is the high-performance, local-first prompt dump built for AI engineers who produce more brilliance than they can manage. It’s not just a library; it’s your personal prompt vault with zero latency, full version control, and a "big boss" vibe.

---

## 🚀 Why PromptSack?

Most prompt managers are too "polite." They want you to sign up, wait for a database sync, and click through ten menus just to save a one-liner. 

**PromptSack** says no to that.

- **Local-First, Always:** Powered by IndexedDB (via Dexie). Your prompts stay on your machine. It's faster than your thoughts.
- **Git-Level Versioning:** Stop naming files `final_v2_REALLY_FINAL.txt`. PromptSack tracks every change, fork, and snapshot automatically.
- **Zero Friction:** Toss a prompt in the "Inbox" and deal with it later. Or organize it into "Collections" if you’re feeling organized.
- **Real-Time Sync:** UI state syncing that actually works. Switch views and your workspace follows.

---

## 🛠️ The Tech Stack (The Muscle)

Built with the modern stack of champions:

- **React 19:** The bleeding edge of UI.
- **Vite:** Because life is too short for slow builds.
- **Tailwind CSS 4:** Maximum styling power with zero bloat.
- **Dexie.js:** Local database that doesn't quit.
- **Zustand:** State management that stays out of your way.
- **Framer Motion:** Smooth transitions for that premium feel.

## 📥 Installation

Ready to get the sack? 

1. **Clone it:**
   ```bash
   git clone https://github.com/your-username/prompt-sack.git
   cd prompt-sack
   ```

2. **Fuel it up:**
   ```bash
   npm install
   ```

3. **Ignition:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) and start dumping.
